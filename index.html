// about

The person behind the terminal

I'm a first-year Computer Science & Engineering student who got into security the natural way — breaking things, wondering why they broke, then figuring out how to break them on purpose.

My interest lies in understanding systems at a low level: how packets move, how memory is laid out, how an attacker thinks. I run a home lab on Proxmox where I practice what I learn — isolated networks, vulnerable VMs, and real tooling.

Alongside security, I build software I actually need: a secure study platform, a local AI assistant, and various automation scripts that save me from clicking things repeatedly.

I'm looking for internship opportunities where I can contribute to real security work while continuing to learn from people who've been doing this longer than I have.
